## 環境構築

### 必要なツール

- Node.js
- npm (Node.js をインストール時に自動的にインストールされる)

### 手順

1. リポジトリのクローン

```bash
git clone https://github.com/Kyutech-Algorithms-Group/graph-builder.git
cd graph-builder
```

2. 依存関係をインストール

```bash
npm install
```

3. 開発サーバーの起動

```bash
npm start
```

## デプロイ

```bash
npm run predeploy
npm run deploy
```
